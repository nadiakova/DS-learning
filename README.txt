DS-learning - папка с проектами, выполненными во время учёбы на курсе Data Science

data - папка с csv файлами и картинками, использующимися в проектах

ExchangeRates.csv - данные о курсах валют

Project_1_git.ipynb - файл Jupyter Notebook с выполненным проектом N1 (анализ базы даных соискателей Head Hunter),
оптимизированный для просмотра на Github. Файлы графиков получены ранее и сохранены в папку data

Solar-Power.ipynb - файл Jupyter Notebook, содержащий финальное задание Аттестации по Питону.
В процессе работы используется файл solar.png

churn_nk.ipynb - финальное задание по визуализации на Питоне, анализ базы клиентов банка. Использует графики из папки data

solar.png - файл с графиком для проекта Solar-Power.ipynb, построен в VS code при помощи алгоритма в файле Solar-Power.ipynb.
Сохранён для корректного отображения графика при просмотре на Github

trading_ETH.ipynb - алгоритм проверки инструментов до начала торговли

workspace-code.workspace - сохранённые настройки рабочей области VS code
